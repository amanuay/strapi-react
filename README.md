## This repo consists of a backend(Strapi) and frontend(React) project 
To run the backend dev server
``` bash
cd my-strapi-project
```
then install the dependencies with
``` bash
pnpm i
```
finally, run
``` bash
pnpm run develop
```
***
To run the frontend app dev server
``` bash
cd vite-project
```
then install the dependencies with
``` bash
pnpm i
```
finally, run
``` bash
pnpm dev
```
